This sample is a practice solution of require.js. It is useful for dependency management of js file.
The build folder contains build.bat file, to run it you need to install node.js application which you can download http://nodejs.org/dist/v0.8.20/x64/node-v0.8.20-x64.msi
build.bat file call node.js, which in turns execute your build.js. The optimize out can be update in release folder.
